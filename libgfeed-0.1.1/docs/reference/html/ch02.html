ATOM API reference

Atom —
AtomCategory — Conveys information about a category associated with an entry or feed.
AtomCommon —
AtomContent — Contains or links to the content of the entry.
AtomDate —
AtomEntry — An individual entry, acting as a container for metadata and data associated with the entry.
AtomGenerator — Identifies the agent used to generate a feed, for debugging and other purposes.
AtomID — A permanent, universally unique identifier for an entry or feed.
AtomLink — A reference from an entry or feed to a Web resource.
AtomPerson — Describes a person, corporation, or similar entity (hereafter, 'person').

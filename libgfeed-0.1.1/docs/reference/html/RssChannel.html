Property Details

The "cloud" property

  "cloud"                RssCloud              : Read / Write / Construct

Allows processes to register with a cloud to be notified of updates to the channel, implementing a lightweight publish-subscribe protocol for RSS feeds.

The "copyright" property

  "copyright"            gchararray            : Read / Write / Construct

Copyright notice for content in the channel.

Default value: ""

The "description" property

  "description"          gchararray            : Read / Write / Construct

Phrase or sentence describing the channel.

Default value: "default"

The "docs" property

  "docs"                 gchararray            : Read / Write / Construct

A URL that points to the documentation for the format used in the RSS file.

Default value: ""

The "generator" property

  "generator"            gchararray            : Read / Write / Construct

A string indicating the program used to generate the channel.

Default value: "<a href=\"http://libgfeed.sourceforge.net\">libgfeed v0.0.1</a>"

The "image" property

  "image"                RssImage              : Read / Write / Construct

Specifies a GIF, JPEG or PNG image that can be displayed with the channel.

The "language" property

  "language"             gchararray            : Read / Write / Construct

The language the channel is written in.

Default value: ""

The "last-build-date" property

  "last-build-date"      gpointer              : Read / Write / Construct

The last time the content of the channel changed.

The "link" property

  "link"                 gchararray            : Read / Write / Construct

The URL to the HTML website corresponding to the channel.

Default value: "default"

The "managing-editor" property

  "managing-editor"      gchararray            : Read / Write / Construct

Email address for person responsible for editorial content.

Default value: ""

The "pub-date" property

  "pub-date"             gpointer              : Read / Write / Construct

The publication date for the content in the channel.

The "skip-days" property

  "skip-days"            guint                 : Read / Write / Construct

A hint for aggregators telling them which days they can skip.

Allowed values: <= 127

Default value: 0

The "skip-hours" property

  "skip-hours"           guint                 : Read / Write / Construct

A hint for aggregators telling them which hours they can skip.

Allowed values: <= 16777215

Default value: 0

The "text-input" property

  "text-input"           RssTextInput          : Read / Write / Construct

Specifies a text input box that can be displayed with the channel.

The "title" property

  "title"                gchararray            : Read / Write / Construct

The name of the channel.

Default value: "default"

The "ttl" property

  "ttl"                  guint                 : Read / Write

Time To Live.

Default value: 0

The "webmaster" property

  "webmaster"            gchararray            : Read / Write / Construct

Email address for person responsible for technical issues relating to channel.

Default value: ""

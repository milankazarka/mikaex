RSS API reference
Rss —
RssChannel —
RssChannelItem —
RssImage — Specifies a GIF, JPEG or PNG image that can be displayed with the channel.
RssGuid — A string that uniquely identifies the item (Globally Unique Identifier).
RssTextInput — Specifies a text input box that can be displayed with the channel.
RssCloud — Allows processes to register with a cloud to be notified of updates to the channel, implementing a lightweight publish-subscribe protocol for RSS feeds.
RssCategory — Specify one or more categories that the channel belongs to.
RssSource — The RSS channel that the item came from.
RssEnclosure — Describes a media object that is attached to the item.
ATOM API reference
Atom —
AtomCategory — Conveys information about a category associated with an entry or feed.
AtomCommon —
AtomContent — Contains or links to the content of the entry.
AtomDate —
AtomEntry — An individual entry, acting as a container for metadata and data associated with the entry.
AtomGenerator — Identifies the agent used to generate a feed, for debugging and other purposes.
AtomID — A permanent, universally unique identifier for an entry or feed.
AtomLink — A reference from an entry or feed to a Web resource.
AtomPerson — Describes a person, corporation, or similar entity (hereafter, 'person').
